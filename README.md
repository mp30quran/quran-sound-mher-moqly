القرآن الكريم بصوت الشيخ ماهر المعيقلى - بدون انترنت


القرآن الكريم - تلاوات - ماهر المعيقلى


استمع إلى تلاوات خاشعة بصوت الشيخ ماهر المعيقلي بجودة عالية، مع إمكانية التشغيل المتواصل والانتقال بين السور بسهولة داخل تطبيق بسيط وسريع.

الميزات:

تلاوات كاملة بصوت الشيخ ماهر المعيقلي 

تشغيل متواصل دون انقطاع

واجهة بسيطة وسريعة

تشغيل في الخلفية مع التحكم من الإشعارات

امكانية تحميل السور الكريمة 

يعمل بدون إنترنت 

يحتوى على سورة الأنعام و جزء عم
و يمكن تحميل باقى السور الكريمة
